Aura 디지털 포토 프레임은 구글 포토 API 변경으로 자동 동기화 기능을 잃을 뻔했지만, 구글의 새 Ambient API로 전체 마이그레이션을 진행하면서 계속 구글 포토 앨범과 자동 동기화될 예정이라고 밝혔다. 요약하면 액자가 계속 사진을 보여줄 수 있게 됐다는 소식인데, 제품의 핵심 기능이 생존 기사처럼 보인다는 점에서 이미 꽤 초라하다.

문제의 핵심은 새 사진이 슬라이드쇼에 자동 반영되는 기능이 더 이상 당연하지 않았다는 데 있다. 디지털 액자에서 자동 업데이트를 빼면 남는 것은 전원을 먹는 추억 보관함인데, 제품에서 제품다움을 구성하는 부분이 사라질 뻔했다는 사실은 좀 멍청하기 짝이 없다.

Aura는 이를 해결하기 위해 구글의 새 Ambient API로 '전체 마이그레이션'을 롤아웃 중이라고 한다. 원래 되던 편의가 다시 되게 만드는 일을 대규모 이전 사업처럼 부르는 태도는 인상적이다. 벽에 걸린 액자가 아니라 서버 정책 바람에 흔들리는 풍향계에 가깝다.

더 우아한 부분은 수동 1대1 사진 선택이다. iOS 사용자는 구글의 공유 시트로, 안드로이드 사용자는 새 인앱 사진 선택기로 이동한다. 같은 기능을 두 플랫폼에서 다른 통로로 우회시키는 설계는 개판이라기보다는, 사용자가 운영체제별 예절 교육까지 받게 하겠다는 고집에 가깝다.

물론 좋게 말할 수도 있다. 자동 동기화가 유지되니 사용자가 일일이 사진을 추가하는 비효율은 줄어든다. 가전이 사용자 대신 일을 하는 척이라도 계속하게 된 셈이니, 최소한 액자 앞에서 수동 노동을 시키는 노골성은 조금 덜해졌다.

하지만 이번 소식이 드러낸 건 더 단순하다. Aura의 액자는 사진을 보여주는 물건이면서도, 그 기본 동작이 구글 API 개편 한 번에 흔들릴 수 있었다. 가전이라기보다 임대 계약 갱신 문자에 생사가 달린 세입자처럼 보인다는 말이 싫다면, 다음에는 적어도 '자동'이라는 단어가 구조 소식처럼 들리게 만들지는 말아야 한다.
